# Branch Plan: epic/7-featured-links-block

## Goal

Add a `mega-menu-featured-links` block that displays a styled, sortable list of links inside the mega menu.

---

## Decisions

- Component uses `"use client"` — justified by `useId()` and `useMegaMenuContext()` hooks
- `MegaMenuContext` stub added locally — will be replaced once mega-menu-editor PR (#1543) merges
- Block returns `null` from save — dynamic server-rendered block pattern
- Link text color is hardcoded to `text-black` — intentional, theming utility planned for next quarter
