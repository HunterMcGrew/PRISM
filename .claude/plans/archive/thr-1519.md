# Plan: Skill Opus Tightening

## Ticket

https://linear.app/tractru/issue/THR-1519/tighten-all-skill-files-and-agentsmd-for-opus-compliance

## Goal

Tighten all 10 skill files and AGENTS.md for Opus compliance: enforce mandatory rule adherence, add scope boundaries, personality sharpening, context window handoff checks, and gap discovery obligations — all additive, no removals.

---

## Decisions

- Mandatory Compliance is a behavioral constraint ("follow rules that apply"), not a file-loading directive — does not cause skills to pull in all architect files
- Clove's AC sync check uses the plan's AC Sync Log, not a direct Linear API call — plan is the source of truth
- Winston and Sasha keep their existing strict "no code" rules — the user's proposed 3-line exception is intentionally rejected to avoid ambiguity
- Personality updates MUST stay within or below current line counts — compress existing text to make room, never add net lines
- Clove's personality shifts to "whimsical dev fairy, professional office" — wonder and craft language, not fantasy roleplay
- Briar's personality shifts to "dark-room Red Bull hunter with Jinx energy" — electric, talks to code, gleeful about catching bugs, controlled chaos, still professional
- Mira's branch check is a soft nudge toward Nora, not a hard gate
- Context window handoff uses heuristics, not a precise % measurement — free, no tool calls
- Context Window Handoff Check MUST only apply to persona/skill handoffs — MUST NOT apply to git operations, questions, or trivial tasks
- Handoff check requires 2 of 3 signals to fire (was: any 1 of 3) — single signals are insufficient evidence of context pressure
- Handoff thresholds: 5+ skill invocations (was 2+), 30+ files read or 1,000+ insertions+deletions per `git diff --stat` (was 15+ files / 500-line diff), 100+ user exchanges (was 10+)
- 0 signals = MUST NOT mention context load at all — prevents unnecessary noise
- Eric MUST always run in a fresh chat — this is a review quality decision, not a context concern. Fresh eyes = unbiased review
- Briar is the gatekeeper for the Eric handoff — her clean-review closing MUST recommend Eric in a new chat unconditionally
