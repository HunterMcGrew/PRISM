# Plan: THR-1573

## Ticket

https://linear.app/tractru/issue/THR-1573/header-column-widths-regressed-to-304030-should-be-402040

## Goal

Fix the mega menu editor header column widths that regressed from 40/20/40 to 30/40/30 during git conflict resolution.

---

## Decisions

- Column widths are 40% / 20% / 40% (left / center / right) — the center column holds only the "Mega Menu" title and needs the least space.
- Center column uses `display: none` with a `@media (min-width: 1025px)` breakpoint to show as flex — matches the `lg` breakpoint convention used elsewhere in the project.
- Fix is SCSS-only — no TSX changes needed since the markup structure is correct.

---

## History
