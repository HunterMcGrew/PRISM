# Plan: THR-1598

## Ticket

https://linear.app/tractru/issue/THR-1598/mega-menu-reset-button-tooltip-and-documentation

## Goal

Update the mega menu editor reset button tooltip text to "Reset Mega Menu" and add the reset button to the editor toolbar documentation.

---

## Decisions

- Change both `label` and `aria-label` on the reset button to "Reset Mega Menu" — the `label` prop drives the tooltip via `showTooltip`, and `aria-label` is the accessible name. Both must match so sighted and screen reader users get the same information.
- Do not change the ResetModal confirmation text — the modal's longer explanation ("Reset to Default Template") is contextually appropriate inside the modal. The tooltip is a short identifier; the modal explains the action. These serve different purposes.
- Update both user docs (`docs/content/user/blocks/mega-menu.md`) and dev docs (`docs/content/dev/architecture/mega-menu.md`) — the user doc's toolbar table is missing the reset button entirely, and the dev doc's toolbar list omits it.
