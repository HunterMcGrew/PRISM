# Branch Plan: robertsamuel/thr-1301-scrolling-banner-editor-refactor

## Goal

Refactor the Scrolling Banner block's editor experience — scope dynamic CSS per block instance, simplify gap calculation, add memoization, and guard attribute setters.

---

## Decisions

- Editor CSS is scoped via `#${clientId} .imageScroller` selectors — prevents style leaking between multiple banner instances in the same editor
- `gapValueMap` lookup replaced with `gap * 0.25rem` formula — works for any numeric value, eliminates `undefined` risk
- `<style>` block guarded with `id ?` check (Option A) — no styles emitted when `id` is absent, eliminates global fallback
- `setAttributes` calls guarded with `!== undefined` checks — prevents resetting attributes when RangeControl fires with undefined
- `EDITOR_HEIGHT_BUFFER_PX` constant extracted from magic number `25`
