## Goal

Fix the "Back to WordPress Admin" button in the Mega Menu Editor header so the browser recognizes it as a link, enabling right-click link options and middle-click / Ctrl+click navigation.

---

## Decisions

- Use `href` prop instead of `onClick` on the `@wordpress/components` Button — `href` causes Button to render as `<a>`, which gives the browser native link context (right-click menu, middle-click, Ctrl+click). No `handleBackToAdmin` wrapper needed.
