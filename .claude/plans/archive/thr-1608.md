# Plan: THR-1608

## Ticket

https://linear.app/tractru/issue/THR-1608/mega-menu-navigation-item-update-help-text-max-items-from-8-to-7

## Goal

Update the mega menu editor sidebar help text threshold from 8 to 7 navigation items and improve its visual prominence by replacing the plain paragraph with a WordPress `Notice` component.

---

## Design

**Status:** Ready for implementation
**Author:** Pixel
**Date:** 2026-04-16

### Summary

Replace the custom `Heading` ("Tips") + `<p className="text-xs text-gray-500">` pattern with a `<Notice status="info" isDismissible={false}>` component from `@wordpress/components`. This is a zero-new-pattern restitch — the `Notice` component is already used in `mega-menu-renderer/edit.tsx` and `carousel-item/edit.tsx`. Update the number from 8 to 7 in the text.

### Decisions worth knowing

- Dropped the "Tips" heading entirely — the Notice visual treatment communicates "this is guidance" without a label (Nielsen #8, aesthetic and minimalist design)
- Used `status="info"` (not `warning`) — this is guidance, not an error or alert. Blue reads as helpful, not alarming.
- `isDismissible={false}` — this guidance should always be visible; dismissing it defeats the purpose
