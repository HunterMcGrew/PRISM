## Decisions

- Gravity Forms migrates from storage zip (`gravityforms_2.9.31.zip`) to `gravity/gravityforms` via `composer.gravity.io`. Reason: official upstream repo, version bumps via `composer update`.
- Version constraint: `^2.10` â€” allows 2.10.x updates, blocks major bump.
- `backend/auth.json` is not checked in â€” credentials are injected per environment: `GRAVITY_FORMS_KEY` secret in CI, env var locally (consumed by docker-compose â†’ Dockerfile ARG â†’ inline `COMPOSER_AUTH`).
- Password field for `composer.gravity.io` http-basic auth is the site URL (`https://tractru.com`), matching the ACF Pro convention already used in the codebase.
